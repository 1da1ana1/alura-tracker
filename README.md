# TaskTracker

TaskTracker é um rastreador de tarefas e tempo desenvolvido com foco em produtividade, experiência do usuário e código modular.  
O aplicativo permite registrar tarefas, iniciar e parar a contagem de tempo, mantendo um histórico visual de forma clara e prática. Também oferece modo escuro para maior conforto durante o uso prolongado.

## Demo Online

Acesse o projeto e interaja com ele online: [TaskTracker](https://alura-tracker-teal-alpha.vercel.app/)

## Funcionalidades

- Registrar tarefas com nome personalizado
- Iniciar e parar contagem de tempo por tarefa
- Visualizar histórico de tarefas em tela
- Alternar entre modo claro e modo escuro
- Interface responsiva e intuitiva

## Tecnologias e Skills Aplicadas

- **Linguagens:** TypeScript, JavaScript (ES6+)
- **Front-end:** Vue.js (Vue 3), HTML5, CSS3, Bulma
- **Ferramentas de Build:** Vue CLI, Node.js, NPM/Yarn
- **Arquitetura:** Component-Based Architecture para reuso e modularização
- **Boas práticas:** Interfaces reativas, design responsivo, manutenção e escalabilidade do código
